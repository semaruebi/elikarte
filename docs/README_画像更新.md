# 精鋭画像の追加方法 💉

ウチが手順を説明するわよ！

## 📸 画像を追加する手順

### 1. 画像ファイルを追加

`assets/images/eliteenemies/` フォルダに画像を配置

**ファイル名の形式：**
```
アイコン_[精鋭名].[拡張子]
```

**例：**
- `アイコン_ファデュイ・氷蛍術師.jpg`
- `アイコン_遺跡守衛.webp`
- `アイコン_王者.jpg`

### 2. 画像リストを自動更新

以下のいずれかの方法で実行：

#### 方法A: Windowsバッチファイル（簡単）
```
scripts/update-elite-images.bat
```
をダブルクリック

#### 方法B: コマンドライン
```bash
node scripts/update-elite-images.js
```

### 3. 確認

- `js/elite-enemy-images.js` が更新されたことを確認
- ブラウザでリロード（Ctrl + Shift + R）
- 精鋭選択モーダルで新しい画像が表示されることを確認

## 🎯 ポイント

- **曖昧マッチング対応**: スプレッドシートの精鋭名と完全一致しなくても、部分一致で自動マッピング
- **複数形式対応**: `.jpg`, `.png`, `.webp` すべてOK
- **自動生成**: 画像リストは自動生成されるので、手動編集不要

## 💡 トラブルシューティング

### 画像が表示されない

1. **ファイル名を確認**: `アイコン_` プレフィックスが必須
2. **日本語文字**: 正しい日本語になっているか確認
3. **スクリプト再実行**: `update-elite-images.bat` を実行
4. **キャッシュクリア**: Ctrl + Shift + R でハードリロード
